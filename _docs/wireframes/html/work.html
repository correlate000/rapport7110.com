お仕事内容

障がいや体調に合わせ、ご自分のペースで働けます。生活リズムの調整と安定、向上ができます。
作業をすることで体力と仕事のスキルが身に付きます。
スタッフと情報共有や意思の疎通を行い、信頼関係を築くことで、
社会活動で大事な、コミュニケーション能力、スキルが高められます。

就労支援（通所）でできること

写真：ガーデニング

ガーデニング・農作業

多肉植物などの育成・販売や、季節の野菜づくりを行います。土に触れながら、植物の成長を感じられるやりがいのある仕事です。

写真：アパレル制作

アパレル制作

オリジナルの商品制作を行います。支援員と一緒に進めます。手芸やパッチワークなど豊富なメニューをご用意しております。

写真：PC作業

PC作業

SNSやネットショップの運営などを行います。パソコンスキルを身につけながら、室内で安定して働けます。

在宅就労でできること

通所にご不安がある方や、ご事情により通所が難しい方は、在宅での作業も可能です。
