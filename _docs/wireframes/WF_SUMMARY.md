# WFサマリー

## 基本方針

**リニューアルではなく「拡張」**
- 既存のTOPページ構成・テキストはそのまま維持
- 新規要素の追加と、下層ページへの導線追加のみ
- セクション順序を最適化（ユーザー導線を優先）

---

## TOPページ構成（最適化済み）

| # | セクション | 対応 | 備考 |
|---|-----------|------|------|
| 1 | KV | 既存そのまま | ファーストビュー |
| 2 | リード | 既存そのまま | 施設の説明導入 |
| 3 | バーベナについて | 既存 + ご利用案内リンク追加 | 詳細説明 |
| 4 | お仕事内容 | 既存 + 詳細リンク追加 | 既存テキスト維持 |
| 5 | 事業所案内 | 既存 + 入り方追加 | アクセス情報 |
| 6 | お問い合わせ | 既存（CTA位置へ移動） | 電話＋フォームリンク |
| 7 | お知らせ | 既存（下部へ移動） | 補足情報として |
| 8 | 採用情報 | 既存そのまま | 別ターゲット向け |

### セクション順序変更の理由
- **メイン導線を優先**：施設を知る → 仕事を見る → 場所を確認 → 問い合わせ
- **NEWSを下部へ**：更新情報は補足。メイン導線を遮らない位置へ
- **CONTACTをOFFICE後に**：「行ってみようかな」と思ったタイミングでCTA

### 追加要素

1. **バーベナについて**
   - 「ご利用案内を見る」ボタン追加

2. **お仕事内容**
   - 「お仕事内容を詳しく見る」ボタン追加

3. **施設への入り方**（事業所案内セクション末尾）
   - 写真3枚で入り方を案内

## クライアント確認事項

### スタッフ紹介ページ
- 掲載するスタッフの人数
- 各スタッフの情報：
  - 役職・名前
  - 趣味
  - 好きな食べ物
  - 休日の過ごし方
  - ひとこと
- 写真の用意（顔出しOK？）

### お仕事内容ページ
- 作業一覧：農園・梱包・清掃の3つでOK？他にある？
- スケジュール：9:00〜15:00で正確？
- 工賃：時給300円〜で正確？載せてOK？

### ご利用案内ページ
- 昼食：提供あり？金額は？
- ロッカー：ある？
- 定期面談：実施している？
- 送迎：対応あり？

### 採用情報ページ
- 給与：金額を確認

### お問い合わせページ
- 受付時間：平日9:00〜15:00で正確？
