## 2. 現状の課題

### 2-1. コンテンツ面

| 課題 | 詳細 |
|------|------|
| 作業内容が不明確 | 農園、梱包など具体的な作業イメージが伝わらない |
| 1日の流れがない | 利用者の1日がイメージできない |
| 写真がフリー素材中心 | 実際の雰囲気が伝わりにくい |
| 施設の強みが不明確 | 他施設との差別化ポイントが見えない |

### 2-2. 情報設計面

| 課題 | 詳細 |
|------|------|
| アクセスの詳細不足 | 駅から遠い、2階にある等の情報が不足 |
| 不安解消コンテンツなし | FAQ等で事前に疑問を解消できない |
| 回遊導線がない | 1ページ完結のため深掘りできない |

---

## 3. リニューアル方針

### 基本コンセプト

```
「比較検討するユーザーが、しっかり見渡せるサイト」
```

- LP型 → **回遊型**へ変更
- 情報を増やし、各ページで詳しく説明
- 不安要素は事前に伝えて期待値調整

## 5. 各ページの改善・追加内容

### 5-1. TOPページ（リニューアル）

**追加・変更するセクション**

| セクション | 内容 | 目的 |
|-----------|------|------|
| 施設の特色 | 3〜4つの強みをアイコン付きで | 差別化 |
| 作業内容プレビュー | 各作業を写真付きで紹介 | 興味喚起 |
| 1日の流れ（簡易版） | タイムライン形式で概要 | イメージ形成 |
| お知らせ | 既存継続 | - |
| アクセス | 地図+「詳しい行き方」リンク | 導線追加 |

### 5-2. 作業内容ページ（拡充）

**現状**: お仕事内容セクションのみ

**リニューアル後**:

| ページ | 内容 |
|--------|------|
| 作業一覧 | 全作業の概要、各詳細ページへの導線 |
| 農園作業 | 作業内容、写真、身につくスキル |
| 梱包作業 | 取扱商品（フェルト、多肉植物、犬グッズ等）、作業風景 |
| 1日のスケジュール | タイムライン形式で1日の流れを視覚化 |

### 5-3. ご利用案内（新規）

| コンテンツ | 内容 |
|-----------|------|
| ご利用の流れ | STEP1〜3形式（見学→体験→利用開始） |
| よくある質問（FAQ） | 不安解消・期待値調整のためのQ&A |

**FAQ例**:
- Q. 駅から遠いですか？ → 正直に「バス15分+徒歩2分」と回答
- Q. 建物のどこにありますか？ → 「2階にあります。入り方は...」
- Q. 週何回から利用できますか？ → 「週1回、半日からOK」

### 5-4. アクセスページ（拡充）

**現状**: 住所と地図のみ

**追加内容**:

| 項目 | 内容 |
|------|------|
| 施設への詳しい行き方 | 写真付きで入口〜2階までを案内 |
| 各駅からのアクセス | 小田急線、南武線、田園都市線 |
| 周辺情報 | 目印になる建物など |
