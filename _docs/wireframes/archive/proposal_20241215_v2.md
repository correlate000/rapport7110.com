## 1. 前提の整理

### 1-1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| 目的 | 広告運用での集客に耐えうるサイトへ改善 |
| 予算感 | 20万円＋α |
| 写真 | 別途対応済み |
| CMS | WordPress（既存継続） |

### 1-2. 広告運用を前提とした設計方針

```
広告 → サイト訪問 → 情報収集 → 問い合わせ
```

この導線を**シンプルかつ確実に**機能させる。

**重要なのは**：
- ページ数を増やすことではない
- 必要な情報が整理されていること
- 不安が解消され、問い合わせに繋がること

## 2. 提案パターン

### パターン比較

| 項目 | A: 既存強化型 | B: ページ追加型 |

---

## 3. パターンA: 既存強化型（推奨）

### 3-1. 概要

現在の1ページ構成を維持しつつ、**コンテンツを強化**する。

広告からの流入を考えると、1ページで完結する方が：
- 離脱が少ない
- 情報が一覧できる
- 運用・更新が楽

### 3-2. 改修内容

| セクション | 現状 | 改修内容 |
|-----------|------|----------|
| KV | キャッチコピー画像 | 必要に応じて差替 |
| お知らせ | 3件表示 | 現状維持 |
| バーベナについて | 概要説明 | **施設の特色・強みを追加** |
| お仕事内容 | 作業紹介 | **作業詳細を拡充、写真差替** |
| 事業所情報 | 住所・地図 | **アクセス詳細を追加**（入り方説明） |
| 採用情報 | 現状 | 現状維持 |
| お問い合わせ | フォーム | 現状維持 |

### 3-3. 追加コンテンツ案

#### A. 施設の特色セクション（新規追加）
- 3〜4つの強みをアイコン+テキストで
- 例：「週1回からOK」「駅からバス送迎」等

#### B. 作業内容の拡充
- 各作業（農園、梱包等）の説明を追加
- 撮影済み写真への差替

#### C. アクセス詳細の追加
- 建物への入り方（写真付き）
- 2階への案内
- ※ネガティブ要素の事前説明

#### D. FAQ追加（任意）
- よくある質問を3〜5個
- 不安解消用

### 3-4. 作業項目と見積もり

| 作業項目 | 工数目安 |
|----------|----------|
| 施設の特色セクション追加 | 0.5日 |

| 写真差替・調整 | 0.5日 |

## 4. パターンB: ページ追加型

### 4-1. 概要

情報量が多い場合、下層ページを追加する。

### 4-2. ページ構成

```
TOP（リニューアル）
├── 作業内容（新規）
│   └── 各作業の詳細、1日のスケジュール
```

## 5. 推奨と判断基準

### パターンAを推奨する理由

1. **予算に収まる**（20万円以内）
2. **広告導線としてシンプル**（1ページ完結）
3. **工数が少なく早く公開できる**
4. **運用が楽**（更新箇所が限定的）

### パターンBが適するケース

- 作業内容が非常に多く、1ページに収まらない
- 将来的にSEOでの集客も狙いたい
- 予算に余裕がある
