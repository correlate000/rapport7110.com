- **回遊型**のページ構成

## 2. ページ構成

### 2-1. 構成図

```
TOP（リニューアル）
│
├── 作業内容
│   └── 作業一覧 + 1日のスケジュール
│
├── ご利用案内
```

## 3. 各ページの内容

### 3-1. TOPページ

| セクション | 内容 |
|-----------|------|
| KV | キャッチコピー + メインビジュアル |
| 施設の特色 | 3〜4つの強みをアイコン付きで |
| 作業内容（概要） | 各作業の紹介 → 詳細ページへ誘導 |
| お知らせ | 最新3件 |
| アクセス（概要） | 地図 + 詳細ページへ誘導 |
| お問い合わせ導線 | CTAボタン |

**削除・移動**:
- 事業所情報の詳細 → アクセスページへ
- 採用情報 → フッターにリンクのみ or 削除検討

### 3-2. 作業内容ページ

| コンテンツ | 内容 |
|-----------|------|
| 作業一覧 | 農園、梱包、その他（写真+説明） |
| 1日のスケジュール | タイムライン形式 |
| 身につくスキル | 任意 |

### 3-3. ご利用案内ページ

| コンテンツ | 内容 |
|-----------|------|
| ご利用の流れ | STEP形式（見学→体験→利用開始） |
| 対象者 | どんな方が利用できるか |
| FAQ | よくある質問 3〜5個 |

**FAQ例**:
- Q. 週何回から利用できますか？
- Q. 見学はできますか？
- Q. 交通費はどうなりますか？

### 3-4. アクセスページ

| コンテンツ | 内容 |
|-----------|------|
| 地図 | Googleマップ |
| 各駅からのアクセス | 小田急線、南武線、田園都市線 |
| 施設への入り方 | 写真付きで案内（建物入口〜2階） |
| 事業所情報 | 住所、TEL、営業時間 |

**ポイント**: ネガティブ要素（駅から遠い、2階）を正直に伝えつつ、行き方を丁寧に説明

## 8. 補足：ページ追加の技術対応

### WordPress側の対応
- 固定ページテンプレートの追加
- ナビゲーションメニューの設定
- 既存テーマ（rapport）をベースに拡張

### 既存テーマの状態
- カスタムテーマ「rapport」使用中
- front-page.php（TOPページ）は1ページ完結型
- 固定ページ用のpage.phpは簡素な状態

→ 各ページ用のテンプレート追加が必要
